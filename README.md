# My Flutter To-Do App

A simple MVVM-style Flutter To-Do application using:
- Riverpod for state management  
- Dio + DummyJSON REST APIs  
- Freezed for immutable models  
- GoRouter for navigation  
- Material 3 theming  
- Unit & widget tests

## Getting Started

1. Clone this repo  
2. cd myapp  
3. flutter pub get  
4. flutter run
